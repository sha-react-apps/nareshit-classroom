Libraries
=========
React Core
----------
1. reactjs( cdn link to include react js library in a particular page)
    https://unpkg.com/react@17/umd/react.development.js
2. Install by using npm: npm install react --save


React Dom
---------
1. CDN Link: https://unpkg.com/react-dom@17/umd/react-dom.development.js
2. Install by using npm: npm install react-dom --save

Babel(compiler)
---------------
1. CDN Link: https://unpkg.com/@babel/standalone/babel.min.js
2. Install by using npm: npm install babel-core --save-dev
